# 숫자 야구 게임
## 기능 목록
* 랜덤 숫자 발생 객체 생성
    * 클래스 생성
    * 랜덤 숫자 발생 객체 테스트 추가
  
* 정답 객체 생성
  * 클래스 생성
  * 테스트 추가
    
* 사용자 입력 받기 클래스 생성
  * 사용자 입력 받는 객체 생
  * 사용자 입력 유효성검증객체 인터페이스 생성
  * 베이스볼 넘버 검증 객체 생성
  * 엔드 넘버(마지막에물어보는) 검증 객체 생성
    
* 게임진행 클래스 생성

## 진행 방법
* 숫자 야구 게임 요구사항을 파악한다.
* 요구사항에 대한 구현을 완료한 후 자신의 github 아이디에 해당하는 브랜치에 Pull Request(이하 PR)를 통해 과제를 제출한다.
